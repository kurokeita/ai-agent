# Technology Stack

## Core

- **Programming Language**: [TypeScript](https://www.typescriptlang.org/) (Strictly enforced)
- **Runtime**: [Node.js](https://nodejs.org/) (Version 25 or higher)
- **Package Manager**: [pnpm](https://pnpm.io/) (Used for dependency management and scripts)

## CLI Framework

- **CLI Driver**: [Commander.js](https://github.com/tj/commander.js) (Standard for CLI command definition and options)
- **Interactive Prompts**: [@clack/prompts](https://github.com/natemoo-re/clack) (Modern and interactive CLI experience)

## Utilities

- **File System**: [fs-extra](https://github.com/jprichardson/node-fs-extra) (Enhanced filesystem operations)
- **Visuals & Colors**: [picocolors](https://github.com/alexeyraspopov/picocolors) (Minimalist and high-performance color output)

## Build and Development

- **Execution**: [tsx](https://github.com/privatenumber/tsx) (Direct execution of TypeScript files)
- **Compilation**: [TypeScript](https://www.typescriptlang.org/) (Compiler for final production build)
- **Linting & Formatting**: [Biome](https://biomejs.dev/) (Unified tool for linting and formatting)
- **Release Automation**: [semantic-release](https://github.com/semantic-release/semantic-release) (Automated versioning and publishing)
